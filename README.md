# What is this about?

Light Cube Engine, is a personal video game engine written in C++ with the aim of learning the modern architecture of 3D rendering engines.

# Modules

- Math
- Serialization
- Audio
- Video
- 2D/3D Rendering
- Memory Optimizations patters
- Kernel system
- Scene system
- Task system
- Entity system
- Component system
